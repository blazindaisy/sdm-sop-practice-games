# SDM & SOP Practice Games

Interactive learning games for child welfare practitioners working with Structured Decision Making (SDM) and Safety Organized Practice (SOP).

## How to use

1. Download or clone this repository to your computer
2. Open `index.html` in any web browser (Chrome, Firefox, Safari, Edge)
3. No installation, internet connection, or technical setup required

## Games

### SOP Jeopardy
A Jeopardy-style game covering Safety Organized Practice principles — harm and danger statements, network building, children's voice, and field application. Designed for use during or after SOP training sessions.

**Features:**
- 5 categories × 4 questions ($100–$400)
- Facilitator reveals answers and awards points by player name
- Live scoreboard — only players who answer appear (no zero scores shown)
- Scores and board state persist if the page is refreshed
- "New Game" button resets everything
- Built-in question editor — customize questions without touching code

## For facilitators

These games are designed for **virtual facilitation with screen sharing**. The facilitator opens the game in their browser, shares their screen, and controls the board while participants call out answers.

## Adding your own questions

Click **Edit Questions** in the top-right corner of any game. You can update category names, questions, and answers directly in the browser — no coding required. Changes are saved automatically and persist across sessions.

## Built with

Plain HTML, CSS, and JavaScript — no frameworks or build tools required.
